## Deploy TygerPyro on VPS
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ sudo apt install python3-venv
 $ git clone https://github.com/Xnuxer2/TygerPyro
 $ cd TygerPyro
 $ python3 -m venv venv
 $ source venv/bin/act*
 $ pip3 install -U -r req*
 $ cp sample.env .env
```
